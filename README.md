# laracasts-testing-vue
A repository in which I store code that was made while following https://laracasts.com/series/testing-vue.

## Setup
### If you have npm installed
`npm install`
### If you have docker installed
```
chmod +x ./install
./install
```

## Running tests
### If you have npm installed
`npm test`
### If you have docker installed
```
chmod +x ./install
./runtests
```
